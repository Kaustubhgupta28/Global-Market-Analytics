# Global Market Analytics Dashboard

## Project Overview

Global Market Analytics is an end-to-end Data Science and Machine Learning project that analyzes global stock market indices and provides predictive insights using multiple ML techniques.

The project includes:

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Market Direction Prediction
* Volatility Prediction
* Anomaly Detection
* LSTM-based Time Series Forecasting
* Interactive Streamlit Dashboard

---

## Dataset

The dataset contains historical data from multiple global stock market indices, including:

* NYA
* IXIC
* HSI
* NSEI
* N225
* GDAXI
* KS11
* SSMI
* TWII
* And others

Features include:

* Open
* High
* Low
* Close
* Volume
* Daily Return
* Moving Averages
* Volatility

## Machine Learning Modules

### Direction Prediction

Predicts whether the market is likely to move upward or downward.

### Volatility Prediction

Estimates future market volatility using historical patterns.

### Anomaly Detection

Identifies unusual market behavior and outlier events.

### LSTM Forecasting

Uses Long Short-Term Memory (LSTM) neural networks for future price forecasting.
